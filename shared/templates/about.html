About Xinu Web

Xinu Web is an experiment designed to negate the typical problems with kernel development (infrastructure, costs, hassle). Though Xinu is not a full-fledged kernel, it is built with educational value in mind.

This web site leverages various technologies to make it possible to boot the kernel in your browser. Most notably is the Javascript PC Emulator built (and copyright) by Fabrice Bellard.

Beyond that, Xinu Web uses Django, jQuery, Bootstrap, Amazon EC2, and PostgreSQL. The actual Xinu kernel is release under a 3-point BSD-style license and the toolchain used to build it is built using crosstool-NG.

This is an on-going experiment. Long term goals include, but are not limited to:

  • Debugging, disk, and network support (to varying degrees)
  • Open source Javascript PC Emulator
  • Other open source Javascript Emulator (ARM, MIPS)
  • Use existing infrastructures to permit booting a raw hardware remotely
